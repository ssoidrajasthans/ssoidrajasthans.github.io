SSO ID Rajasthan Portal

Single Sign-On (SSO) ID Rajasthan is a unified digital identity that allows citizens, businesses, and government employees to access multiple Rajasthan government services using one secure login.

What is SSO ID Rajasthan?

SSO ID Rajasthan is an initiative of the Government of Rajasthan aimed at simplifying access to online public services. Instead of remembering multiple usernames and passwords for different departments, users can log in using a single SSO ID.

The official portal sso.rajasthan.gov.in integrates various departments and services, promoting transparency, efficiency, and paperless governance across the state.

Types of SSO ID Rajasthan

  • Citizen SSO ID: For residents of Rajasthan to access welfare schemes and citizen services.
  • Udhyog SSO ID: For business owners, industries, and entrepreneurs.
  • Government Employee SSO ID: For Rajasthan state government employees.

SSO ID Rajasthan Login Process

  1. Visit the official website sso.rajasthan.gov.in
  2. Enter your SSO ID and Password
  3. Fill in the captcha code
  4. Click on the Login button
  5. Access all linked Rajasthan government services

How to Register for SSO ID Rajasthan

  1. Open the SSO Rajasthan portal
  2. Click on Register
  3. Select user type (Citizen / Udhyog / Government Employee)
  4. Register using one of the following:
    • Jan Aadhaar
    • Bhamashah Card
    • Aadhaar OTP
    • Google Account
  5. Create your SSO ID and password

Services Available on SSO Portal

Frequently Asked Questions (FAQs)

Q1. Is SSO ID mandatory?
Yes, SSO ID is mandatory for most Rajasthan government online services.

Q2. Is SSO ID free?
Yes, SSO ID registration and usage are completely free.

Q3. Can I reset my SSO password?
Yes, you can reset your password using the Forgot Password option.
