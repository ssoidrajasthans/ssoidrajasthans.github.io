SSO ID Rajasthan Portal

Single Sign-On (SSO) ID enables citizens, businesses, and government employees to access multiple Rajasthan government services using one secure login.

What is SSO ID Rajasthan?

SSO ID Rajasthan is a digital identity system launched by the Government of Rajasthan. It allows users to access various online services with a single username and password.

The official portal sso.rajasthan.gov.in promotes transparency, digital governance, and paperless service delivery.

Types of SSO ID

  • Citizen SSO ID – For residents of Rajasthan
  • Udhyog SSO ID – For businesses and entrepreneurs
  • Government Employee SSO ID – For Rajasthan state employees

SSO ID Rajasthan Login Steps

  1. Visit sso.rajasthan.gov.in
  2. Enter your SSO ID and Password
  3. Complete captcha verification
  4. Click on Login

How to Register for SSO ID Rajasthan

  1. Open the SSO Rajasthan portal
  2. Click on Register
  3. Select user type
  4. Register using:
    • Jan Aadhaar
    • Bhamashah Card
    • Aadhaar OTP
    • Google Account
  5. Create SSO ID and password

Services Available on SSO Portal

  • e-Mitra Services
  • Jan Aadhaar
  • Scholarship Portal
  • Employment Services
  • ePDS (Ration Card)
  • Pay Manager
  • IFMS & HRMS
  • Business Registration
  • Digital Certificates
  • Rajasthan Sampark

Frequently Asked Questions

Q1. Is SSO ID mandatory?
Yes, SSO ID is mandatory to access most Rajasthan government online services.

Q2. Can I reset my SSO password?
Yes, use the Forgot Password option on the login page.

Q3. Is SSO ID free?
Yes, SSO ID registration is completely free.
